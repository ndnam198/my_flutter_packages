# reusable_bloc_base

Light-weight extension library for flutter_bloc

## Features

Cover most of the common work flow in a flutter bloc, avoid boilerplate code.

## Getting started

Install the package

```zsh
    flutter pub add reusable_bloc_base
```

## Usage

```dart
const like = 'sample';
```
